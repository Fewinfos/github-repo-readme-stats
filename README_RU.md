# 📦 GitHub Repository Stats Widget

Открытое, полностью клиентское средство визуализации **статистики GitHub-репозитория в реальном времени** в интерактивном и настраиваемом формате — идеально для разработчиков, мейнтейнеров и создателей портфолио.

---

## 🎯 Цель

Виджет использует GitHub REST API для получения и отображения метаданных и аналитики любого публичного репозитория. Работает **напрямую в браузере** — не требуется сервер или аутентификация.

---

## ✨ Возможности

- 🔄 Получение данных в реальном времени через GitHub REST API
- ⭐ Отображает звёзды, форки, наблюдателей, issues и pull-запросы
- 👥 Показывает топ-контрибьюторов с аватарами и количеством коммитов
- 📊 Графики используемых языков
- 📅 Дата создания и последнего обновления
- 📜 Информация о лицензии
- 🎨 Чистый, адаптивный и настраиваемый интерфейс
- 💻 Работает прямо в браузере (сервер не нужен)
- 🧩 Легко встраивается в сайты и README
- 📈 Дополнительные графики через Chart.js

---

## 🧱 Технологии

- **HTML** – структура
- **CSS** – стили и адаптивность
- **JavaScript** – логика и API
- **GitHub REST API** – источник данных
- **Chart.js** – визуализация графиков (опционально)

---

## 📊 Доступные виджеты

### 🔍 Статистика репозитория

- ⭐ Звёзды / 🍴 Форки / 👁️ Наблюдатели
- 📅 Дата создания и обновления
- 📜 Тип лицензии
- 📊 Используемые языки (круговые, столбчатые диаграммы)
- 📦 Зависимости (npm, pip и т.д.)
- 📈 Активность коммитов (тепловая карта)
- 🕐 Среднее время объединения PR
- 🧵 Статусы Issues (открытые / закрытые / закреплённые)

### 👥 Виджеты контрибьюторов

- 👥 Топ контрибьюторы (аватары + коммиты)
- 📊 Коммиты по дням недели
- 🗺️ Карта контрибьюторов (публичные данные)
- ⏱️ Недавние контрибьюторы (7 / 30 дней)
- 📈 Вклад во времени (график)

### 📊 Графические виджеты

- 📊 Радар-диаграмма «здоровья» репозитория
- 📉 График роста звёзд/форков
- 🍩 Donut-график языков
- 📈 Динамика Issues/PR (область)
- 📆 GitHub-стиль календаря коммитов

### ⚙️ DevOps и CI/CD

- 🚦 Статус GitHub Actions
- 🧪 Покрытие кода (Codecov, Coveralls)
- 🔄 Последний запуск workflow
- 🛠️ Хронология билдов

### 📌 Issues и Pull Requests

- 📋 Закреплённые Issues / обсуждения
- 🔍 Облако слов меток
- 📬 Статистика Pull-запросов
- 📈 Анализ настроения по Issues

### 🧩 Дополнительно

- 📌 Кнопка "в избранное"
- 🔍 Поиск других репозиториев
- 🧠 AI-анализ коммитов (опция)
- 🔗 Связанные репозитории
- 🪄 Экспорт как iframe / HTML

## 🚀 Развёртывание

Виджет можно развернуть через **GitHub Pages** или любой хостинг для статичных сайтов: Netlify, Vercel, Firebase.

### Развёртывание через GitHub Pages

1. Отправьте проект на GitHub
2. Перейдите в **Settings → Pages**
3. Выберите ветку: `main` и папку: `/ (root)`
4. Виджет будет доступен по адресу:  
   `https://ваш_логин.github.io/github-repo-stats-widget/`
